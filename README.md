# Heart Rate Monitor

This project is a heart rate monitor that measures heart rate using a MAX30102 sensor and displays the results on an SSD1351 OLED display.

## Hardware
- MAX30102 sensor
- SSD1351 OLED display
- Raspberry Pi 4B

## Credits
This project uses changed version of MAX30102 library by [Doug Burrell](https://github.com/doug-burrell).
